# Phase 8.5 (단계 ②a) — 메인 라이브러리 뷰 + 모드 토글 설계

> 작성일: 2026-06-30 · 대상: cmd-docu · 우선순위: 티어 3 (Phase 8.5의 두 번째 단계)
> 관련: PRD §3.9, CLAUDE.md Phase 8.5. 선행: 단계 ①(사이드바 PARA 렌즈, `ParaLens`) 완료.
> 범위: ②의 첫 슬라이스(②a) — **썸네일 제외**(종류별 아이콘 셀). 썸네일은 ②b, 폴더별 뷰 기억은 ③.

## 1. 목적

"파일 하나 읽는 리더" 위에 "폴더를 격자/리스트로 훑는 라이브러리" 모드를 더한다.
PARA를 출력(라우팅)뿐 아니라 **탐색 축**으로 쓴다. 첨부(이미지·PDF·오피스)를
텍스트의 곁다리가 아니라 그 자체로 본다. 읽기/탐색 전용 — 파일 변경 없음.

## 2. 확정된 결정 (brainstorming)

- **범위**: ②a = 모드 토글 + LibraryView(그리드/리스트) + `selectedFolder` + 클릭 동선.
  셀은 **종류별 SF심볼 아이콘 + 이름**. 썸네일(QLThumbnailGenerator+NSCache)은 ②b.
- **폴더 선택**: Finder식 — 폴더 **행(라벨) 탭 = 선택**(selectedFolder 설정 + 라이브러리 전환),
  disclosure **삼각형(chevron) = 펼침 전용**. → `FileTreeItemRow`를 커스텀 행으로 리팩터.
- **모드 전환**: 마지막 동작이 이김. 폴더 탭→library, 파일 클릭→reader, 툴바 토글→그 모드.
  별도 충돌 로직 없음.
- **레이아웃 기억**: 전역 1개(`libraryLayout`). 폴더별 기억은 ③.
- **경계**: 읽기/탐색 전용. 이동·삭제·이름변경 없음. 모델 불변(렌더/순수만 + 상태 추가).

## 3. 컴포넌트 & 상태

### 신규 상태 (`AppState`)
- `var mainMode: MainMode = .reader`
- `var selectedFolder: URL?` — 라이브러리가 보여줄 폴더. 기본·리셋값은 `currentFolder`.
- `var libraryLayout: LibraryLayout = .grid`

### 신규 모델 `Sources/Models/LibraryMode.swift`
- `enum MainMode: String, Codable { case reader, library }`
- `enum LibraryLayout: String, Codable { case list, grid }`

### 신규 서비스 `Sources/Services/LibraryListing.swift`
- `static func entries(of folder: URL) -> [FileTreeItem]`
  - `folder`의 **직속 children**(파일 + 1단계 하위폴더)을 `FileTreeItem`으로. 숨김 제외.
    하위폴더는 `isDirectory == true`, 파일은 `false`. 정렬은 호출부에서 `ParaLens.sorted`.
    접근 불가/없음이면 빈 배열(크래시 없음).

### 신규 뷰 `Sources/Views/LibraryView.swift`
- 헤더: `selectedFolder` 이름 + 상위로(↑) 버튼(`currentFolder` 위로는 안 감).
- 본문: `ParaLens.sorted(LibraryListing.entries(of: selectedFolder), under: currentFolder)`를
  `libraryLayout`에 따라 `LazyVGrid`(grid) 또는 `List`(list)로.
- 셀: 종류별 SF심볼 아이콘(`FileTreeItem.icon`) + 이름. `ParaLens.classify(_:under:)`가
  `.archive`면 `.opacity(0.45)`(렌즈 일관).
- 폴더 셀 클릭 → 드릴인(`selectedFolder = 그 폴더`, library 유지).
- 파일 셀 클릭 → `openDocument(at:inNewTab:)` + `mainMode = .reader`.
- 빈/무선택: `selectedFolder ?? currentFolder`가 nil이면 "폴더를 여세요" 안내.

### 수정 `MainEditorView.swift`
- `mainMode == .library`면 리더 디스패치(image/pdf/office/markdown/Welcome) 대신 `LibraryView()`.

### 수정 `ContentView.swift` (툴바)
- 새 **모드 세그먼트** `[리더 | 라이브러리]`(`mainMode` 바인딩).
- 보조 토글 모드별 교체: reader → 기존 `ViewModePicker`(Source/Split/Preview),
  library → `[리스트 | 그리드]`(`libraryLayout`).

### 수정 `SidebarView.swift` (`FileTreeItemRow`)
- `DisclosureGroup` → 커스텀 행: 명시 chevron 버튼(펼침/접힘만) + 라벨 `onTapGesture`
  (폴더면 `selectFolderForLibrary(item.url)`, 파일이면 기존 `openDocument`).
- 단계 ① PARA 렌즈 스타일(projects 강조·archive dim 라벨 한정)·즐겨찾기 별·컨텍스트 메뉴 보존.
- 자식 `ForEach(ParaLens.sorted(item.children, under: currentFolder))` 유지.

## 4. 데이터 흐름 & 상호작용

```
[사이드바] 폴더 행 탭  → selectFolderForLibrary(url): selectedFolder=url, mainMode=.library
            삼각형 탭   → toggleFolderExpansion(url) (모드·선택 불변)
            파일 행 탭  → openDocument(url): mainMode=.reader
[툴바]      모드 세그먼트 → mainMode = 선택값
            보조 토글     → reader: viewMode / library: libraryLayout
[라이브러리] 폴더 셀     → selectedFolder = 하위폴더 (library 유지)
            파일 셀     → openDocument + mainMode=.reader
            헤더 ↑      → selectedFolder = 상위(>= currentFolder)
[열기]      openFolder/currentFolder 변경 → selectedFolder = currentFolder (mainMode 유지)

LibraryView 렌더: entries(of: selectedFolder ?? currentFolder)
  → ParaLens.sorted(_, under: currentFolder)  → LazyVGrid/List (셀=아이콘+이름, archive dim)
```

### AppState 메서드
- `func selectFolderForLibrary(_ url: URL)` — selectedFolder=url, mainMode=.library.
- `func enterReader()` 또는 기존 openDocument 경로에서 `mainMode = .reader` 설정.
- currentFolder를 바꾸는 기존 지점(openFolder/loadFileTree 진입)에서 `selectedFolder = currentFolder`.

## 5. 에러 처리 (크래시 없음)

- 빈 폴더 → 빈 그리드 + "이 폴더에 항목이 없습니다".
- selectedFolder·currentFolder 모두 nil → "폴더를 여세요" 안내.
- 접근 불가 폴더 → `entries`가 빈 배열, 안내 표시.
- 탭 없음 + reader → 기존 WelcomeView.

## 6. 테스트 전략

순수·상태 단위테스트(뷰·툴바는 수동):
- `LibraryListingTests`(임시 디렉터리): 직속 폴더+파일 반환, 숨김 제외, 하위폴더 isDirectory·파일 not,
  빈 폴더 → 빈 배열, `ParaLens.sorted`와 결합 시 PARA 순서(폴더먼저·archive 끝).
- `AppLibraryStateTests`(@MainActor): 기본값(mainMode=.reader, libraryLayout=.grid, selectedFolder=nil);
  `selectFolderForLibrary(url)` → selectedFolder 설정 + mainMode=.library; 파일 열기 → mainMode=.reader;
  currentFolder 변경 → selectedFolder 리셋.
- 사이드바 행(chevron=펼침/라벨=선택): 상태 메서드는 위에서 단위테스트, 행 자체는 빌드+수동.
